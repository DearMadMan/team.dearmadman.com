Eloquent: 序列化

简介

当构建 JSON APIs 时，你需要经常的将你的模型和关联转换为数组或者 JSON 格式。Eloquent 包含了便捷的方法来进行这些转换，以及控制哪些属性应该包含在序列化之中。

基础用法
