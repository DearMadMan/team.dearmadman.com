如何解码合约的事件日志

Posted on 2017-12-13

Event

合约中启用事件有三个好处:

  • 可以主动通知函数执行的结果，正常情况下发送一笔交易只会返回这比交易的 Hash 信息，而不是执行结果
  • 可以进行事件监听，与 UI 层进行交互
  • 可以节省 Gas, 因为相对于在 Contract 中进行直接存储，执行事件所消耗的 Gas 更小 (Contract 20,000 gas / 32 bytes, logs 8 gas / byte)
